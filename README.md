[AlloyPhoto](http://alloyteam.github.com/AlloyPhoto/) - Image Processing Lib Based On HTML5
===========================

### Fetures

1. 基于html5的JS图片图像处理库
2. 简单快捷的API
3. 多图层操作
4. 与PS对应的17种图层混合模式
5. 多种基本滤镜处理效果
6. 多种组合风格效果
