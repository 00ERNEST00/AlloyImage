Welcome to AlloyPhoto.

AlloyPhoto是一个基于html5技术的JS库，集成了一些方便快捷的图像处理API，您可以将它引用到您的网页中，结合css3等技术，做出一些之前无法做到的优美效果，或者，提供一个用JS编写的在线图像处理工具等。

特点

1.图层功能，提供图层的添加，删除，交换图层顺序等功能，且包含与PS相对应的17种图层混合模式
2.图像的基本调节功能，包括亮度、对比度，色相、饱和度、明度调节
3.多种滤镜功能，去色、反相、高斯模糊、锐化、浮雕效果、查找边缘、马赛克、腐蚀等
4.处理后文件的保存，处理完成之后，可以将文件输出为base64形式间接使用和保存
5.高级组合效果，如素描，lomo，hrd，复古，素描，油画(功能尚未添加） 如一个素描效果的实现

var img = new Image();
img.src = "pic.jpg";
img.onload = function(){
      //素描效果
      psLib(this).act("灰度处理").add(//添加一个图层上去
            psLib(this).act("灰度处理").act("反色").act("高斯模糊",5) , "颜色减淡"  
      ).act("锐化").show();
};

API参考

初始化一个psLib对象

psLib(img) 或 new psLib(img)
img可以是new Image()创建的对象，也可以是DOM树中的image元素，但在创建psLib对象前确保 img已经被onload成功，否则可能导致抛出异常

var img = document.getElementsByTagName("img")[0];
img.onload = function(){
     var psLibObj = psLib(this);
};

一个psLib对象可以看做是一个图层，也可以看做是一个工作文件

act

说明：对单个图层进行效果处理
语法：psLibObj act(String method,Args arg,...)
其中 method有以下可选参数

参数 功能 附加参数 用法示例
亮度 调节亮度与对比度 亮度，对比度 act("亮度",10,20)
色相/饱和度调节 调节图像色相与饱和度 色相，饱和度，明度 act("色相/饱和度调节",10,0,10)
灰度处理 将图像处理为灰度图像 无 act("灰度处理")
灰度阈值 将图像二值化 阈值 act("灰度阈值",128)
反色 将图像进行反相操作 无 act("反色")
高斯模糊 将图像进行高斯模糊 模糊大小 act("高斯模糊",10)
锐化 将图像进行锐化 锐化大小 act("锐化",10)
马赛克 将图像进行马赛克处理 马赛克大小 act("马赛克",10)
浮雕效果 将图像进行浮雕效果处理 无 act("浮雕效果")
腐蚀 将图像进行腐蚀效果处理 无 act("腐蚀")

add

说明：合并一个图层到本图层上去
语法：psLibObj add(psLibObj new[,String model [,Number dx,Number dy]])
本方法将会进行图层合并操作，会将新图层new以高于本图层的顺序，以model的混合模式，以相对本图层左上角为坐标原点的dx,dy偏移量合并到本图层，并返回合并后的图层对象
model的模式有17种，其效果和Photoshop中效果对应
"正常","颜色减淡","变暗","变亮","正片叠底","滤色","叠加","强光","差值","排除","点光","颜色加深","线性加深","线性减淡","柔光","亮光","线性光","实色混合"
